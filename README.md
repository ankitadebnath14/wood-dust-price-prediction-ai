# 🌲 AI-Based Wood Dust Price Prediction System

A Machine Learning web application that predicts wood dust prices using factors like:

- Wood Type
- Moisture
- Quantity
- Transport Cost
- Season
- Market Demand

## 🚀 Features

- AI-powered price prediction
- Interactive Streamlit dashboard
- Business insights
- Data visualization
- Machine Learning model training

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## 📊 Machine Learning Model

Random Forest Regressor

## ▶ Run Locally

```bash
streamlit run app/app.py
```
